## Resumen
En este repositorio se albergan los materiales didácticos, las investigaciones y los avances de la colección de Ciencias de la Tierra como parte de la biblioteca PlanBiologia FCUNAM, desarrollada a través del proyecto PAPIME Competencias para la enseñanza-aprendizaje de bioliteratura: Una intervención estratégica.
